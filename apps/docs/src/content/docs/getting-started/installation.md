---
title: Installation
description: Install Minions Orders packages
---

## TypeScript / Node.js

```bash
npm install @minions-orders/sdk minions-sdk
```

## Python

```bash
pip install minions-orders
```

## CLI

```bash
npm install -g @minions-orders/cli
```

## Requirements

- **Node.js** 20 or later
- **Python** 3.11 or later (for Python SDK)
