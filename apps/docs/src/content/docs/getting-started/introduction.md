# Minions Orders

Order records, service selections, onboarding checklists, and invoices

## Overview

This project is part of the [Minions ecosystem](https://github.com/mxn2020/minions) and provides:

- **TypeScript SDK** (`@minions-orders/sdk`) — Core library for Node.js
- **Python SDK** (`minions-orders`) — Python bindings
- **CLI Tool** (`@minions-orders/cli`) — Command-line interface
- **Web App** — Interactive playground at [orders.minions.wtf](https://orders.minions.wtf)
